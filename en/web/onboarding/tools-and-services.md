## Communication Tools

<details markdown="1">
  <summary><strong>Install Outlook and Teams as PWA (Recommended)</strong></summary>

For best performance, install Outlook and Teams as Progressive Web Apps through Microsoft Copilot:

**Outlook PWA:**
1. Open [https://copilot.microsoft.com/](https://copilot.microsoft.com/) and sign in with your work account.
2. In the browser address bar menu, select **Apps → Install this site as an app**.
3. Name it **Outlook** and complete the installation.

**Teams PWA:**
1. Open [https://copilot.microsoft.com/](https://copilot.microsoft.com/) and sign in with your work account.
2. In the browser address bar menu, select **Apps → Install this site as an app**.
3. Name it **Teams** and complete the installation.
