🧠 역할 및 권한 문서

이 프로젝트는 복잡한 기존 그룹 구성을 깔끔하고 유지 관리 가능한 역할/권한 모델로 대체하는 확장 가능한 RBAC 구조를 정의합니다. 기존 시스템의 접근 수준을 유지하면서, 예외 처리·가격 제한·직접 재정의도 지원합니다.

📑 목차

  • 🧠 역할/권한 개요
    역할 카테고리, 기능, 작업, 비즈니스 채널, 접근 수준의 시각적 분해.
  • 📊 RBAC 데이터 모델
    새 역할/권한 구조의 핵심 테이블과 관계 설명.
  • 🧱 테이블 정의
    각 테이블의 컬럼, 관계, 비고를 상세 기술.
  • 🧩 역할 정의
    모든 기본 역할과 해당 권한 수준의 분해.
  • 🔄 마이그레이션 계획
    레거시 그룹을 RBAC 역할로 전환하고 배포하는 단계별 절차.
  • 🧮 권한 매트릭스
    모든 정의된 역할에 대해 기존 권한을 새로운 권한 수준으로 매핑.
  • 🧭 평가 순서
    중첩되는 권한을 우선순위로 어떻게 해소하는지 설명.
  • 🚫 가격 표시 금지 규칙
    특정 예외에서 가격 권한을 제거하는 제한 규칙 정의.
  • 🧾 레거시 매핑
    레거시 그룹을 현대적 역할로 해석하는 방법 문서화.
  • 🧷 직접 권한 부여
    역할 기반 상속 외 사용자별 재정의 가이드.
