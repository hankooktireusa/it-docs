🧠 역할 및 권한 문서

이 프로젝트는 복잡한 기존 그룹 구성을 깨끗하고 유지 관리 가능한 역할과 권한 구조로 대체하는 확장 가능한 RBAC(역할 기반 액세스 제어) 구조를 정의합니다. 가격 제한, 직접 재정의, 예외 상황을 지원하며, 기존 시스템에서 제공하던 동일한 수준의 액세스를 보장합니다.

📑 목차

  • 🧠 역할 및 권한 개요
    역할 범주, 기능, 작업, 비즈니스 채널, 접근 수준의 시각적 구조도.
  • 📊 RBAC 데이터 모델
    새로운 역할 및 권한 구조의 핵심 테이블과 관계를 설명합니다.
  • 🧱 테이블 정의
    각 테이블의 컬럼, 관계, 참고 사항에 대한 상세 스키마.
  • 🧩 역할 정의
    모든 기본 역할과 해당 역할에 할당된 권한의 세부 목록.
  • 🔄 마이그레이션 계획
    기존 그룹을 RBAC 역할로 변환하고 새 시스템에 배포하는 단계별 절차.
  • 🧮 권한 매트릭스
    모든 정의된 역할에 대해 기존 권한을 새로운 권한 수준으로 매핑합니다.
  • 🧭 평가 순서
    중복된 권한이 우선순위에 따라 어떻게 처리되는지 설명합니다.
  • 🚫 가격 표시 제한 규칙
    특정 예외 상황에서 가격 관련 권한을 제거하는 제한 규칙 정의.
  • 🧾 레거시 매핑
    기존 그룹이 현대적인 역할로 해석되는 방법을 문서화.
  • 🧷 직접 권한 부여
    역할 기반 상속 외에 사용자별 재정의를 부여하는 가이드.
