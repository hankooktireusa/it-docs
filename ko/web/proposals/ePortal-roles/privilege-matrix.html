📊 권한 매트릭스

역할 전반에 걸쳐 사용되는 모든 권한과, 각 권한이 노출하는 권한 수준을 보여주는 전체 매트릭스입니다.

ℹ️ 참고: 이 매트릭스는 새로운 RBAC 역할 모델에서 정의된 권한 수준을 반영합니다.
아래 나열된 각 권한은 role-definitions 에 설명된 역할에서 사용되는 방식에 따라 A, S, U, L 권한과 매핑됩니다.
이 문서는 향후 감사를 지원하고, 재정의 로직과 역할 설계 결정을 지원하기 위한 것이며, 과거 그룹 사용 이력을 반영하지 않습니다.

📑 목차 (클릭하여 펼치기)
  • Order
  • National Account
  • Status
  • Report
  • Warranty
  • All Permissions

A = 접근(Access)
S = 재고(Stock)
U = 단가(Unit Price)
L = 정가(List Price)
