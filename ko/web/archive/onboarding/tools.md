## 커뮤니케이션 도구

### Copilot을 통한 PWA(Progressive Web App) 설치

- **Outlook PWA** / **Teams PWA**
  1. [Copilot](https://copilot.microsoft.com/)에서 업무 계정으로 로그인
  2. 브라우저 메뉴 → **앱 → 이 사이트를 앱으로 설치**
  3. 이름 지정 후 설치 완료
